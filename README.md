# AERIONN

A titanium carry-on, reconsidered. Coming Spring 2026.

Brand landing page — typography-led, editorial grid, Swiss poster design language.

## Stack

Single-file HTML + vanilla JS + SVG. No build step.

- Type: Inter (thin wordmark) / Space Grotesk (display) / Space Mono (meta)
- SVG filter-based horizontal motion-blur on display type
- Scroll-linked parallax, velocity-driven blur, custom cursor
- 12-column vertical grid, edge-to-edge horizontal rules

## Develop

Open `index.html` in any browser. No dependencies.

## Files

- `index.html` — the site
- `aerionn-mark.svg` — brand mark
- `aerionn-lockup.svg` — full lockup (mark + wordmark + tagline)
- `favicon.svg`
- `og-image.svg` — social share card
